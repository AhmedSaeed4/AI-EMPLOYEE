# Update Handbook

Add new rules, guidelines, or procedures to the `AI_Employee_Vault/Company_Handbook.md` file.

## What This Does

1. Reads the current Company_Handbook.md
2. Adds new rules or updates existing ones
3. Maintains proper formatting and structure
4. Tracks changes with timestamps

## Usage

```
/update-handbook
```

Then describe what you want to add or change.

## Example

```
/update-handbook

Add a rule: All payments over $100 require approval from two people.
```

## Handbook Sections

| Section | Purpose |
|---------|---------|
| Core Principles | Fundamental behavior guidelines |
| Communication Guidelines | Email, WhatsApp, social media rules |
| Financial Rules | Payment and invoice handling |
| Task Priorities | How to prioritize work |
| Approval Workflow | When to ask for human approval |
| Things AI Should NEVER Do | Safety boundaries |

## Notes

- Changes are logged with timestamps
- Handbook should be reviewed weekly during CEO Briefing (Gold tier)
- All AI behavior should follow these rules
